# 🌿 Tea & Hibiscus Leaf Disease Detection using Deep Learning

## 📖 Project Description

This project is an AI-powered web application developed to detect diseases in **Tea** and **Hibiscus** leaves using Deep Learning. The system enables users to upload a leaf image and instantly identifies whether the leaf is **healthy** or affected by a specific disease. The application is designed to support farmers, researchers, and agricultural professionals by providing fast, reliable, and accessible disease detection.

The deep learning model is trained on a dataset containing approximately **2,000 images** of Tea and Hibiscus leaves, covering multiple disease categories as well as healthy leaves. The trained model is integrated into a **Streamlit** web application, allowing users to perform real-time predictions through a simple and user-friendly interface.

### 🚀 Features

* Real-time leaf disease detection
* Supports Tea and Hibiscus plants
* Image upload for instant prediction
* Displays predicted disease class
* Shows confidence score for each prediction
* Lightweight and easy-to-use interface
* Suitable for desktop and mobile web browsers

### ▶️ How to Run

1. Clone the repository.
2. Install the required dependencies:

```bash
pip install -r requirements.txt
```

3. Start the Streamlit application:

```bash
streamlit run app.py
```

4. Open the generated local URL in your browser.
